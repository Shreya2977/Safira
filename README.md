# 🛡️ Safira – Safe. Secure. Seen.

> A Preventive Safety Companion for Safer Urban Travel

---

## 📖 Overview

Safira is a proactive safety companion mobile application designed to enhance women's safety during travel, especially at night.

Unlike traditional safety apps that rely on manual SOS triggers, Safira focuses on **continuous monitoring, intelligent check-ins, and preventive reassurance** — ensuring users feel safe throughout their journey, not just during emergencies.

---

## 🚨 Problem Statement

Women frequently feel unsafe while traveling alone at night.

### Current Challenges

- Most existing safety apps are **reactive** (panic/SOS button based).
- No **continuous journey monitoring**.
- Emergency systems activate **only after danger is perceived**.
- Lack of preventive reassurance during travel.
- Alerts depend on manual user action.

---

## 🔎 How Safira Is Different

| Existing Solutions | Safira |
|--------------------|--------|
| Manual panic button | ✅ Automatic intelligent monitoring |
| Reactive approach | ✅ Preventive & proactive approach |
| Emergency-only focus | ✅ Focus on *before-danger phase* |
| User must trigger alert | ✅ Alerts triggered even if user is unresponsive |

Safira shifts the safety model from reaction to prevention.

---

## 🧠 How It Solves the Problem

- ⚡ Reduces reaction time through automation  
- 👀 Ensures continuous journey awareness  
- 🔄 Performs automatic responsiveness checks  
- 🚨 Triggers alerts if user becomes unresponsive  
- 📍 Shares last known location & route  
- 🔐 Maintains privacy with trip-based access control  

---

## 🚀 Features

- 📍 Live GPS route tracking
